# Funcionalidades

 - 1 - Unauthorized
 - 2 - Login
 - 3 - Cadastro
 - 4 - logout
 - 5 - Refresh token JWT
 - 6 - Listar Infomaçãoes do usuario logado
 - 7 - Listar Barbeiros favoritos
 - 8 - Favoritar Barbeiro
 - 9 - Listar Agendamentos do usuario
 - 10 - Listar todo os barbeiros
 - 11 - Listar barbeiro pelo ID
 - 12 - Fazer agendamento com o barbeiro
 - 13 -Buscar barbeiro

# Rotas

 - 1 - /401
 - 2 - /auth/login
 - 3 - /user
 - 4 - /auth/logout
 - 5 - /auth/refresh
 - 6 - /user
 - 7 - /user/favorites
 - 8 - /user/favorite
 - 9 - /user/appointments
 - 10 - /barbers
 - 11 - /barber/{id}
 - 12 - /barbers/{id}/appointment
 - 13 - /search

2- Para iniciar a aplicação:
> php artisan serve
